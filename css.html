¿Para qué sirven estos agrupadores?
<thead>: cabecera de tabla, permite estilo y accesibilidad.
<tbody>: cuerpo de la tabla, ayuda a organizar visual y semánticamente los datos.
<tfoot>: pie, suele mostrar totales, notas, etc.
TIP: Úsalos para colorar o diferenciar zonas con CSS, así puedes aplicar el mismo estilo a varias tablas.

Glosario web útil

  • BLOQUE: Ocupa todo el ancho, inicia en nueva línea (ej.: div, section, article)
  • EN LÍNEA: No inicia línea nueva, solo ocupa el ancho necesario (ej.: span, a, img)
  • display: Cambia cómo se comporta (block, inline, flex, grid...)
  • Pseudoelementos: ::before, ::after: añaden contenido decorativo
  • Variables CSS: var(--n): valores personalizados para temas y utilidades
Practica: Cambia, borra o mueve bloques para ver cómo se transforman las etiquetas visuales y los colores.

Variables y funciones modernas

:root {
  --base: #98C4E1;
  --sombra: 0 2px 8px #a7e6cf66;
}
.ejemplo-box {
  background: var(--base);
  box-shadow: var(--sombra);
  font-size: clamp(1em, 2vw, 2.2em);
  width: calc(100vw - 3em);
}
¿Por qué usar variables (--nombre)? Te permiten cambiar todo el diseño desde un solo sitio, crear modos oscuro/claro fácilmente, y evitar repetir colores o medidas.
¿Para qué sirven calc(), clamp(), min(), max()? Permiten diseñar layouts realmente adaptables y fluidos, sin necesidad de decenas de media-queries.

Tips visuales y prácticas

  • Utiliza colores y esquemas suaves para diferenciar bloques. Hazlo en tu propio proyecto para ver la estructura rápidamente.
  • Nunca uses sólo colores para transmitir información (accesibilidad: combina color y etiquetas/descripciones).
  • Elige display:flex/grid para maquetar. Usa selectors modernos :not(), :nth-child(), ::before para depurar y experimentar rápido.
  • Consulta la documentación oficial de propiedades para aprender detalles avanzados.
  • ¡No dudes en copiar y pegar estos fragmentos en tus propios proyectos para analizarlos y adaptarlos!
