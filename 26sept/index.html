a)Qué especificidad tienen los siguientes selectores:

* -> tiene de especificidad 0,0,0,0 porque es un selector universal
.rojo -> tiene de especificidad 0,0,1,0 porque es un selector de clase
.autos.seat -> tiene de especificidad 0,0,2,0 porque son dos selectores de clase
#autos.rojo -> 110 por id+class
#autos .rojo -> 110 por id+class
#autos > * > .verde -> 110 por id+universal+class
p#autos > * > * > span -> 102 por elemento+id+universal+universal( dos veces )+element
p -> 1 por elemento
div p -> 2 por elemento+elemento
div.autos p -> 12 por elemento+class+elemento
div#coches p span -> 103 por elem+id+elemento+elemento
table -> 1 por elemento
td -> 1 por elemento
table tr td -> 3 por elemento ( tres veces )
table tr td + td -> 4 por elemento ( cuatro veces )
td ~ td -> 2 por elemento ( dos veces )
table.roja tr -> 12 por elemento+class+elemento
table#calles -> 101 por elemento+id
table[title] -> 11 por elemento+atributo
table[title][lang] -> 21 por elemento+atributo+atributo
:nth-child(3) -> 10 por pseudo-clase
:last-child -> 10 por pseudo-clase
td:nth-child(3) -> 11 por elemento+pseudo-clase
table td:nth-child(3) -> 12 por elemento+elemento+pseudo-clase
table tr td:nth-child(3) -> 13 por elemento+elemento+elemento+pseudo-clase
table.ciudades tr td:nth-child(3) -> 23 por elemento+class+elemento+elemento+pseudo-clase
table#calles.ciudades tr td:nth-child(3) -> 123 por elemento+id+class+elemento+elemento+pseudo-clase
