Manual Detallado de CSS

Guía estructurada con teoría, ejemplos y comentarios. Inspirada en los contenidos de dww.es/css y documentación moderna.

0. ¿Qué es CSS?

CSS (Cascading Style Sheets) es un lenguaje que permite separar la presentación del contenido en una web. Permite controlar colores, fuentes, márgenes, posicionamiento, animaciones y adaptabilidad de la página.

1. Sintaxis y selectores

  • Cada regla CSS tiene un selector y un bloque de propiedades:

.selector {
  propiedad: valor;
  /* Comentario: esta línea aplica 'valor' a 'propiedad' en elementos coincidentes */
}
  • Selectores básicos: etiqueta (p), clase (.clase), id (#id)
  • Selectores combinados: descendiente (div p), hijo directo (nav > a), adyacente (h1 + p)
  • Selectores avanzados: [type="text"], :hover, ::before, :nth-child(odd)
odede>
ul.menu li:first-child { font-weight:bold; }
a:hover { color:red; } /* Al pasar mouse */
input[type="email"] { border:2px solid steelblue; }

2. Inclusión de CSS

  • En línea (dentro de la etiqueta HTML): odede><h1 style="color:green;">Hola</h1>
  • En la cabecera HTML (interno): 
    odede>
      <style>
      body { background:#f1f1f1; }
      </style>
  • Archivo externo (profesional): 
    
      <link rel="stylesheet" href="estilos.css">

3. Colores y Unidades

  • Colores: nombre (red), hex (#e63946), rgb (rgb(200,0,0)), rgba (con transparencia), hsl.
  • Unidades: px, %, em, rem, vw/vh (viewport), ch, fr (Grid).

body { color: #222; background: rgb(251,251,248); }
div { width: 30em; height: 20vh; }

Usa rem/em para escalabilidad y adaptabilidad. px es fijo; % relativo al contenedor.

4. Fuentes y Texto

  • odede>font-family: fuente o grupo (odede>Arial, Verdana, sans-serif)
  • font-size, font-weight, font-style
  • odede>line-height: altura entre líneas
  • text-align: alineación horizontal (left, center, right, justify)
  • letter-spacing, word-spacing, text-transform (mayúsculas/minúsculas)

h1 {
  font-family: 'Segoe UI', Arial, sans-serif;
  font-size: 2em;
  font-weight: bold;
  text-align: center;
  text-transform: uppercase; /* MAYÚSCULAS */
}

5. El modelo de caja (Box Model)

  • Todo elemento tiene: contenido, padding (espacio interno), border (borde) y margin (fuera del borde).
  • box-sizing: content-box (por defecto, el ancho no incluye border/padding) o border-box. 

.caja {
  width: 300px;
  padding: 10px;    /* Relleno hacia dentro */
  border: 4px solid #307;
  margin: 15px auto;
  box-sizing: border-box; /* ancho incluye padding y border */
}

6. Fondos, Bordes y Sombras

  • background-color, background-image, background-size, background-position, background-repeat
  • border, border-radius, box-shadow
odede>
.banner {
  background: url("fondo.jpg") center/cover; /* cover estira, center centra */
  border-radius: 16px;
  box-shadow: 0 6px 30px rgba(40,60,120,.17);
}

7. Posicionamiento y display

  • display: block, inline, inline-block, none, flex, grid, table, etc.
  • position: static, relative, absolute, fixed, sticky
  • float, clear (flujo flotante y limpieza)
  • odede>z-index: superposición de elementos

.header { position: fixed; top: 0; left: 0; width: 100%; }
.avatar { float: right; margin-left:1em; }
.capa-top { z-index: 99; position: absolute; }

8. Selectores avanzados y pseudo-elementos/clases

  • :hover, odede>:active, :focus, :nth-child(n), ::before, odede>::after 

.btn:hover { background: #ffe080; cursor: pointer; }
.lista li:nth-child(odd) { background: #eef; }
.input:focus { border-color: #4d8; }
.alerta::before { content: "⚠️ "; }

9. Responsive: Media queries

  • Permiten adaptar los estilos según el tamaño de pantalla o condiciones del dispositivo.

@media (max-width: 600px) {
  nav { flex-direction: column; }
  h1 { font-size: 1.3em; }
}

10. Flexbox

  • Sistema de cajas flexibles para alinear, distribuir y ordenar en una y dos dimensiones.

.contenedor {
  display: flex;
  flex-direction: row;     /* fila (horizontal) || column (vertical) */
  justify-content: center; /* centra horizontalmente */
  align-items: center;     /* centra verticalmente */
  gap: 1em;                /* separación entre hijos */
}
.item { flex: 1 1 200px; }

11. Grid CSS

  • Diseño bidimensional avanzado: filas y columnas personalizables.

.grid {
  display: grid;
  grid-template-columns: 2fr 1fr;
  grid-gap: 2em;
}
.grid > div {
  background: #eef;
  border-radius: 8px;
  padding: 14px;
}

12. Animaciones y transiciones

  • transition: suaviza cambios de valores
  • animation: define animaciones con @keyframes

.boton {
  transition: background 0.4s, transform 0.3s;
}
.boton:hover {
  background: #aaf;
  transform: scale(1.1);
}
@keyframes latido {
  0%, 100% { transform: scale(1); }
  50% { transform: scale(1.2);}
}
.cora {
  animation: latido 1s infinite;
}

13. Buenas prácticas y consejos

  • Escribe reglas generales antes que específicas, para que puedas sobrescribir.
  • Usa clases para reutilizar estilos.
  • Organiza el CSS en módulos por componente/zona/proyecto.
  • Valida tu CSS y usa prefijos si es necesario (-webkit-, -ms-, -moz- para navegadores viejos).
  • Comenta usando /* ... */ tus reglas importantes.
  • Simplifica el uso de !important salvo para excepciones reales.
  • Siempre piensa en accesibilidad y responsive.

14. Selectores útiles y truco rápido

  • * — selector universal; afecta a todo: * { box-sizing: border-box; }
  • [attribute] — selecciona por atributo: input[type="password"]
  • :not() — aplica donde NO exista: li:not(.activado)
  • :nth-child(n), :first-child, :last-child — afecta al N-ésimo hijo: tr:nth-child(odd)
  • ::before, ::after — añade contenido decorativo sin HTML extra

.btn::before { content: "→ "; color: #42b; }

15. Variables CSS (Custom Properties)

Las variables permiten definir valores reutilizables y cambiar todo el tema visual muy fácil.
Se definen con --nombre y se usan con var(--nombre). 


:root {
  --color-principal: #2a75b3;
  --radius: 10px;
}
.boton {
  background: var(--color-principal);
  border-radius: var(--radius);
  color: white;
}
/* Puedes cambiar el valor en cualquier parte del documento HTML */

16. Propiedades clave agrupadas ("chuleta CSS")

  • Color y fondo: color, background-color, background-image, opacity
  • Texto: font-size, font-family, font-weight, text-transform, text-shadow, letter-spacing, white-space
  • Caja: width, height, min/max-width, overflow, box-shadow, border, border-radius, outline
  • Espaciado: margin, margin-top/right/bottom/left, padding, padding-*
  • Posicionamiento: position, left/top/right/bottom, z-index
  • Flex/Grid: display, flex-direction, flex-wrap, justify-content, align-items, grid-template-rows/columns, gap
  • Transición/Animación: transition, transform, animation, keyframes
  • Responsive: @media, max-width, min-width, orientation
  • Otros: cursor, pointer-events, user-select, visibility, content

17. Glosario esencial (para novatos)

  • Selector: parte antes de las llaves, decide qué etiquetas afectas
  • Propiedad: lo que quieres modificar (color, margen, posición)
  • Valor: cómo quieres la propiedad (rojo, 20px, flex, etc.)
  • Regla: conjunto selector+propiedad-valor (p { color: red; })
  • Herencia: qué estilos "bajan" a los hijos (ej: color texto)
  • Especificidad: qué regla prevalece si varias afectan lo mismo
  • Reset CSS: archivo para igualar todos los estilos por defecto de los navegadores

20. Filtros, blend modes y efectos visuales


.caja-blur {
  filter: blur(4px) brightness(0.6) grayscale(0.8);
  /* Se pueden combinar filtros para hacer difuminados y otros efectos */

Los filtros y blending-modes permiten efectos como fondos borrosos tipo "glassmorphism", superposiciones y creatividad gráfica sin usar Photoshop.

21. Funciones CSS (calc, clamp, min/max)


h1 {
  font-size: clamp(1.3rem, 2.5vw, 3rem); /* Tamaño fluido entre 1.3rem y 3rem */
}
.seccion {
  width: calc(100% - 2em); /* Calcula tamaño restando espacios */
  margin: min(5vw, 24px) auto; /* Toma el valor menor según ancho viewport */
}

Funciones como calc, clamp y min/max permiten ajustarse mejor a dispositivos, hacer layouts responsivos y eliminar muchas media queries.

25. Variables CSS avanzadas, herencia y reacción a media queries


:root {
  --tam-base: 1.2rem;
  --sombra: 0 2px 10px #4af3b650;
}
.caja {
  font-size: var(--tam-base);
  box-shadow: var(--sombra);
}
@media (max-width: 700px) {
  :root {
    --tam-base: 1rem;
  }
}
/* Ahora cambia el tamaño base automáticamente si la pantalla es pequeña */

Así puedes crear temas claros/oscuro, cambiar estilos globalmente por modo móvil, o reaccionar a preferencias de usuario solo cambiando una variable.

27. Especificidad y !important

Especificidad es la "fuerza" de un selector para sobreescribir estilos.
- Etiqueta: 1 punto
- Clase o pseudoclase: 10 puntos
- ID: 100 puntos
- Inline style: 1000 puntos
El !important debe evitarse, pero fuerza una regla sobre todas. 


#autos { color: red; }           /* 100 puntos */
.autos.seat { color: green; }    /* 20 puntos */
div p span { color: blue; }      /* 3 puntos */
* { color: gray !important; }    /* fuerza máxima, sobreescribe TODO si es declarado después */

31. Selectores avanzados (sin class ni id)

  • selector de atributos: input[type="text"]
  • selector hijo directo: article > section
  • selector hermanos: td + td (adyacente), td ~ td (todos posteriores)

33. Tableros, diseños sin clases: ajedrez, cuadrantes y listas


/* Tablero de ajedrez 8x8 */
.chess td { width:2.5em; height:2.5em; text-align:center; font-size:2em;}
.chess tr:nth-child(even) td:nth-child(odd),

En tableros y tablas grandes, usa nth-child combinado y combinaciones de posición para colorear celdas, cuadros o cuadrantes como piden los ejercicios. No uses class ni id, solo selectores.
